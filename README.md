A Wox plugin for Fanfou

## Dependency

```bash
$ pip install fanfou
```

## Usage

1. Config
 
```
ff config consumer_key consumer_secret
```

2. Login

```
ff login username password
```

3. Post

```
ff Hi Wox Fanfou
```
